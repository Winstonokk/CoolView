*****控件分类：
-------

1.可拓展可收缩的ExpandTextView
2.高仿ios的底部多类型菜单AlertView
3.不可滑动的viewPager
4.自己动手实现圆形，圆角图片
5.表情键盘
6.第三方分享底部dialog自定义
